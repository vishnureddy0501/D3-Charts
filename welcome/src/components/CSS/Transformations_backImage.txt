skew: distort an element along the X and/or Y axes by a specified angle. 
        This transformation skews the shape of an element, making it appear slanted.
transform: skew(20deg, 20deg); //slant x-axis and y-axis
transform: skewX(20deg); //slant the x-axis
transform: skewY(20deg);  // slant y-axis
